# Tha Dub Cub (TDC)

A web application for Tha Dub Club gaming community - 

Home page, member accounts, clipboard where members post and vote on clips, and a leaderboard and member profiles 
derived from that activity. 

## Tech Stack
- **Backend:** Python, Flask
- **Database:** SQLite
- **Frontend:** HTML, CSS, Bootstrap 5, Jinja templates
- **Version Control:** Git / Github

## Status
In active development. Currently building user registration and login

## Running locally 
python -m venv venv\
venv\Scripts\activate\
pip install -r requirements.txt\
python app.py\

Then open http://127.0.0.1:5000
